Excel File Format

Your Excel file should have the following columns:

Column Name Required Data Type Range/Values
Student ID Required Text/Number Unique identifier
Name Optional Text Student's full name
Gender Optional Text Male/Female
Age Optional Number 16-30
Attendance % Required Number 0-100
Assignment Marks Required Number 0-100
Midterm Marks Required Number 0-100
Final Exam Marks Required Number 0-100
Study Hours Required Number 0-24
Previous GPA Required Number 0-4
Extracurricular Optional Number 0, 1, 2, 3
Family Income Optional Number USD amount
Parent Education Optional Text High School, Bachelor, Master, PhD
Tips
  • Use consistent data formats
  • Ensure Student IDs are unique
  • Check for missing required fields
  • Validate numeric ranges
  • Remove any empty rows
Important Notes
  • First row should be headers
  • Duplicate Student IDs will be skipped
  • Invalid data will be flagged
  • Processing may take a few minutes
